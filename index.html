Part 1

  • По тегу - свойство применится ко всем тегам на странипце. Например, h1 {color: red}
  • По классу - свойство применится ко всем элементам, с таким классом. Например, .class {color: red}
  • По id - свойство применится ко всем элементам, с таким id. Например, #id {color: red}
